![logo-img](https://res.cloudinary.com/dbadhyp7x/image/upload/v1777378902/Cs2-icon_lc7nll.jpg)
## Description

This landing page delivers an immersive and high-energy experience inspired by the competitive world of Counter-Strike. The design uses a dark, cinematic aesthetic with orange accents, animated transitions, and dynamic section changes that create a smooth, game-like navigation flow. Each section highlights different game modes, maps, and weapon categories through interactive cards and bold visual composition.

A continuous background soundtrack enhances the atmosphere, making the experience feel more engaging and intense. As users scroll or switch between sections, the interface responds with full transitions and motion effects that keep the navigation seamless and visually impactful. The combination of sound design, animated interactions, and gaming-inspired visuals gives the landing page a modern, immersive identity focused on action and competition.

## Stacks

For this project, we used HTML5, CSS3, and JavaScript to build and structure the website, creating an interactive and visually immersive environment. Git and GitHub were used for version control and collaborative development throughout the project. Cloudinary was integrated to manage and optimize media assets efficiently, while Figma was used for interface design, prototyping, and maintaining visual consistency during the design process.

## Methodology

The project was developed following Agile principles through the Scrum framework, allowing the team to organize tasks efficiently and iterate continuously during both the design and development phases. We also adopted the Atomic Design methodology to create a scalable and modular interface system, improving consistency across components and facilitating future expansion of the website.
